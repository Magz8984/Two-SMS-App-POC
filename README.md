### POC Two Way SMS

## Description

User sends sms with ID number to a twilio number and gets registered.

## Author

Collins Magondu Muthinja


## Technology

- Nest JS

## Set Up

- Install nest js cli
- Run npm install
- Run npm run start
- Use ngrok to set up tunnel for app
```ngrok http 3000```

**NB: Twilio number is in trial account**
